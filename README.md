# DigiTools

It is a single page application where user can buy digital products like AI writing tools, Video Editing Software etc.

## Technology 

This project is built using the following technologies:

* Tailwind CSS
* React
* Daisy UI
* Javascript
* HTML

## Key Features

1. Product Browsing : user can browse digital products
2. Cart System : user can add or remove products from cart and checkout
3. Toastify: user will get notifications for adding or removing products from cart
